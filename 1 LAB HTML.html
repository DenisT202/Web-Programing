C language

BACKGROUND

I learned to program in C as a second programming language at university, I was able to create projects like a game in C which helped me to better understand how this programming language works.

Features

We can choose the C language for many reasons:
  • It does not require a lot of resources
  • Data structure the unions of the C language are complex
  • Has access to the memory by using pointers

Comparison

Pros Cons
Highly portable and interoperable No checks for runtime
Rich set of built-in and user-defined functions No provision for having a namespace
Extendable with library functions No concept of a constructor and destructor
